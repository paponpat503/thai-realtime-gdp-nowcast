# Thai Real-Time GDP Nowcasting System

This repository hosts the project structure for the research project **“Thai Real-Time Nowcasting System: A Mixed-Frequency VAR Platform for Real GDP Monitoring.”**

The project aims to build a transparent and reproducible nowcasting framework for Thailand’s real GDP using mixed-frequency data, automatic indicator screening, and modern forecasting techniques.

### ⚠ Code Availability

This repository currently provides the **project structure, documentation, and placeholders only**.  
The full implementation will be released after internal review and project evaluation.

### Project Structure

- src/: Placeholder modules  
- notebooks/: High-level overview  
- data/: Empty directories  
- docs/: Documentation  
- examples/: Minimal example
